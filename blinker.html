Project 01

LED blinker

This project gets an LED blinking on and off so students can learn basic output, wiring, and upload flow.

Back to projects

What you need

  • Arduino board
  • LED
  • 220Ω resistor
  • Breadboard and jumper wires

How it works

  1. Wire the LED and resistor to a digital pin and ground.
  2. Upload a sketch that turns the pin on and off.
  3. Change the delay to see the blink rate change.

Starter sketch

#define LED_PIN 13

void setup() {
  pinMode(LED_PIN, OUTPUT);
}

void loop() {
  digitalWrite(LED_PIN, HIGH);
  delay(1000);
  digitalWrite(LED_PIN, LOW);
  delay(1000);
}
